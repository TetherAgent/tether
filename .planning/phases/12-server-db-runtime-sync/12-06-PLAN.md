
注意：agent.turn payload 中 turnIndex 字段改为 0（占位），Server 侧的 gateway_chat_messages.turn_index
由 Server 在写入时按 session 的 turn 序号自行管理（或按 event.id 作为唯一排序依据）。
实际上 Server runtimeSyncRepository.upsertChatMessage 接收的 turnIndex 来自 event payload，
值为 0 时依然可以幂等写入（UNIQUE session_id + turn_index），因此每次 agent.turn 事件 turnIndex
应使用 event.id 保证唯一性。
修改 emitAssistantTurn / emitUserTurn 时，改为 turnIndex: event.id 而非 0：
```typescript
const event = this.store.appendEvent(this.sessionId, 'agent.turn', {
  role: 'assistant', content, tools, turnIndex: 0  // placeholder，Server 侧按 event.id 区分
});
```
实际 Server 侧 upsertChatMessage 用 event payload.turnIndex 作为 turn_index 写入，
为了不产生重复行（相同 session_id + turn_index），应将 turnIndex 设为 event.id（保证唯一）。
最终 emitAssistantTurn：
```typescript
const event = this.store.appendEvent(this.sessionId, 'agent.turn', {
  role: 'assistant', content, tools, turnIndex: -1  // placeholder
});
// 用 event.id 作为最终 turnIndex，发布前重写 payload（或直接传 event.id）
this.publishEvent({ ...event, payload: { ...event.payload, turnIndex: event.id } });
```
简化方案：event.id 作为 turnIndex（event.id 是 SQLite 自增 ID，在单 session 下单调递增，
符合 UNIQUE session_id + turn_index 约束）：
```typescript
private emitAssistantTurn(content: string, tools: Array<{ name: string; inputSummary: string }>): void {
  const toolsJson = tools.length > 0 ? JSON.stringify(tools) : undefined;
  const event = this.store.appendEvent(this.sessionId, 'agent.turn', {
    role: 'assistant' as const,
    content,
    tools,
    turnIndex: 0  // temporary; overwritten below
  });
  const eventWithTurnIndex = {
    ...event,
    payload: { ...event.payload, turnIndex: event.id }
  };
  this.publishEvent(eventWithTurnIndex);
  this.statusPublisher?.onAgentTurn('assistant');
}

private emitUserTurn(content: string): void {
  const event = this.store.appendEvent(this.sessionId, 'agent.turn', {
    role: 'user' as const,
    content,
    tools: [] as Array<{ name: string; inputSummary: string }>,
    turnIndex: 0  // temporary; overwritten below
  });
  const eventWithTurnIndex = {
    ...event,
    payload: { ...event.payload, turnIndex: event.id }
  };
  this.publishEvent(eventWithTurnIndex);
  this.statusPublisher?.onAgentTurn('user');
}
```
