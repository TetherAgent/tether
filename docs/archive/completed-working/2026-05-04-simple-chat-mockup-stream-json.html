总结apps/web 共有 6 个路由：

  • / — 对外官网首页（公开）
  • /login — 登录页
  • /register — 注册页
  • /sessions — 登录后 session 列表
  • /remote/session/:sessionId — 单个终端 session（高级视图）
  • /remote/session/:sessionId/simple — 简洁聊天视图

所有路由集中在 src/routes.tsx，main.tsx 只负责把 Provider 和 session surface 传给路由层 —— 这与 apps/web/CLAUDE.md 里"路由必须先进入 routes.tsx"的规范是一致的。
